⚠️ How to render your local .pbix file

A .pbix file cannot be read directly by an HTML file. You must host it first. Follow these steps:

  1. Open your file Campaign Performance Marketing_Power BI project by Mohit Bhardwaj_April 25, 2026 (1).pbix in the Power BI Desktop application.
  2. Click Publish on the Home ribbon to upload it to the Power BI Service (Workspace).
  3. Go to app.powerbi.com and open the report.
  4. Go to File > Embed report > Publish to web (public) (or Website/Portal for secure access).
  5. Copy the generated HTML <iframe> code.
  6. Replace the placeholder <iframe> in this index.html file with your code.
